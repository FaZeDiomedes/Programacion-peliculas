🎬 Programación Películas

Plataforma web de películas desarrollada como proyecto de práctica y portafolio, enfocada en diseño moderno, organización del código y experiencia de usuario.

📖 Descripción

Programación Películas es una aplicación web diseñada para explorar y visualizar contenido cinematográfico mediante una interfaz intuitiva y atractiva.

El proyecto fue desarrollado con el objetivo de fortalecer habilidades en:

Desarrollo web
Lógica de programación
Diseño responsive
Organización de proyectos
Uso de Git y GitHub

Además, sirve como base para futuras implementaciones más avanzadas relacionadas con APIs, autenticación y gestión de usuarios.

🚀 Instalación

Clona el repositorio:

git clone https://github.com/FaZeDiomedes/Programacion-peliculas.git

Ingresa a la carpeta del proyecto:

cd Programacion-peliculas

Ejecuta el proyecto abriendo el archivo index.html en tu navegador.
